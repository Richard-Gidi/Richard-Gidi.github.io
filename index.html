Prediction of Insurance Claims

This project demonstrates the process of Exploratory Data Analysis and Predictive Modeling using Python. The goal is to gain insights from the dataset and build predictive models to forecast the target variable.

Food Insecurity in Africa

I conducted a thorough exploratory data analysis (EDA) and developed a machine learning model using Python to predict the severity of food insecurity in Korogocho and Viwandani. Key libraries used include Pandas, Matplotlib, Scikit-learn, and XGBoost to analyze household data and identify at-risk households.

Anime Recommendation System

In this project, I developed an anime recommendation system using a Kaggle dataset. The project involved data preprocessing, feature engineering (e.g., extracting TF-IDF vectors for anime synopses), and building a content-based recommendation system using cosine similarity to match users with relevant anime. The system also features a user-friendly interface for personalized recommendations.

Data Analysis: Food Prices In Ghana

This project involved building an interactive Power BI dashboard analyzing 18 years of food price trends across 10 regions in Ghana. It highlights fluctuations in average prices for cereals, tubers, meats, fruits, and vegetables, with a special focus on rice (local vs imported)

Power BI: Life Expectancy Overview

This project analyzes factors affecting life expectancy by incorporating previously overlooked variables such as immunization rates and the Human Development Index (HDI). The analysis is based on a mixed effects model and multiple linear regression, utilizing data from 2000 to 2015 for 193 countries. The focus will be on significant health-related factors, including immunization (Hepatitis B, Polio, Diphtheria), mortality rates, economic conditions, and social factors.
